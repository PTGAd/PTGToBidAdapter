#ToBid平台，Fancy广告适配器

## 导入 SDK
使用CocoaPods导入SDK

```shell
  pod 'ToBid-iOS','4.4.30'
  pod 'PTGAdFramework','2.2.64'
  pod 'PTGToBidAdapter','1.0.0'
```

## 适配器相关
需要在ToBid后台添加自定义广告平台

### 初始化适配器
PTGToBidConfigAdapter
需要在添加自定义广告源时，需要添加应用维度参数app_id,app_key参数（fancy广告平台的app_id,app_key)
   
### 开屏
PTGToBidSplashAdapter
